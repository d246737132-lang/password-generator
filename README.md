# 生活效率工具箱

一个基于 React + Vite 构建的生活效率工具集合，包含实用的日常工具。

## 🚀 功能特性

### 🔐 密码生成器
- 自定义密码长度（4-64位）
- 支持大写字母、小写字母、数字、特殊字符
- 实时密码强度检测
- 一键复制到剪贴板
- 快捷键支持：Ctrl+G 生成，Ctrl+C 复制

### 🧮 简易计算器
- 支持基本四则运算（+、-、×、/）
- 清除、删除、正负号切换
- 百分比计算
- 键盘快捷键支持
- 计算完成提示

## 🛠️ 技术栈

- **框架**: React 18
- **构建工具**: Vite 5
- **图标库**: Lucide React
- **样式**: 自定义 CSS（深色主题）

## 📦 安装与运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
